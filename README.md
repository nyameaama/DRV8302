# DRV8302
6-Phase Hardware Control of DRV8302

Six-step (trapezoidal) commutation pattern for a three-phase BLDC motor. Each phase voltage is driven to one of three levels —+Vₛ, 0, or –Vₛ— over six equal time‐steps.
